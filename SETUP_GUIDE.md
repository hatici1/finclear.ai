# How to Run FinClear AI

Follow these simple steps to run the app on your computer.

### 1. Install Node.js
You need a tool called **Node.js** to run modern web apps.
- Go to [nodejs.org](https://nodejs.org/)
- Download and install the **LTS Version** (Recommended for most users).

### 2. Download the Code
- Download this project folder to your computer.
- Unzip it if necessary.

### 3. Open Terminal
- **Windows**: Open the folder, right-click in an empty space, and select "Open in Terminal" (or Command Prompt).
- **Mac**: Right-click the folder and select "New Terminal at Folder".

### 4. Install & Run
Type the following command and press Enter to download the required tools:
```bash
npm install
```

Once that finishes (it may take a minute), type this command and press Enter to start the app:
```bash
npm run dev
```

### 5. Open in Browser
You should see a message like `Local: http://localhost:3000`.
- Open your web browser (Chrome, Edge, Safari).
- Go to **http://localhost:3000**

### Optional: Install as App
Once the app is open in Chrome or Edge:
1. Look for an install icon in the address bar (computer screen with a down arrow).
2. Click **Install FinClear AI**.
3. It will now run like a native app on your desktop!
